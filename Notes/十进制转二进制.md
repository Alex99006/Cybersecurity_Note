# 十进制转二进制小抄（8 位表示）

## ✅ 权值表（从左到右）

| 位数 | 1    | 2    | 3    | 4    | 5    | 6    | 7    | 8    |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 权值 | 128  | 64   | 32   | 16   | 8    | 4    | 2    | 1    |

---

## ✅ 转换步骤

以 **168（十进制）** 为例：

1. 找出最大的权值 ≤ 168：128 ✅  
2. 168 - 128 = 40  
3. 下一个能用的是 32：40 - 32 = 8 ✅  
4. 下一个能用的是 8：8 - 8 = 0 ✅  
5. 剩下的权值都用不上，补 0 即可

---

## ✅ 二进制结果

| 位数            | 1    | 2    | 3    | 4    | 5    | 6    | 7    | 8    |
| --------------- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 权值            | 128  | 64   | 32   | 16   | 8    | 4    | 2    | 1    |
| 是否使用（1/0） | 1    | 0    | 1    | 0    | 1    | 0    | 0    | 0    |

🔢 结果：`168（十进制） = 10101000（二进制）`

| ------ | ----------------- |

| 200    | `11001000`        |
| 255    | `11111111`        |
| 0      | `00000000`        |

## 📌 温馨提示：

- 每个 IPv4 地址段（如 192.168.10.39）都是一个 8 位的十进制数（范围是 0~255）
- 每个段都可以单独转换成二进制，网络掩码和子网计算常用！
